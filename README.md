## рџ“ќ Description

A browser-based Tic-Tac-Toe implementation built with modern CSS and vanilla JavaScript. The project entry point is [index.html](index.html) and it relies on native browser APIs rather than heavy frameworks.

---

## вљ™пёЏ Interesting Techniques

* **[Intersection Observer](https://developer.mozilla.org/en-US/docs/Web/API/Intersection_Observer_API)**: Used to detect when the game board enters the viewport, triggering entrance animations without relying on scroll event listeners.
* **[CSS Scroll Snapping](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_scroll_snap)**: Implements a snap behavior on the match history container, allowing users to swipe through past games cleanly.
* **[CSS Custom Properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties)**: Manages theming and responsive scaling from a single source of truth in the stylesheet.
